# Notes for Homework 2

## Goal

The goal of this homework was to explore the syntax and capabilities of the
    Python package `pandas`.

## Overview

The following `pandas` functionalities have been used:

- load in data from a csv file,
- set global viewing options for `pandas` using the `pd.set_option` function,
- store and manipulate string, int, float and datetime values in a `DataFrame`,
- use the `map` method on `Series` objects,
- use the `df.fillna` method to replace NaN values,
- create new columns by assigning them to values with new column names,
- add string-typed columns together,
- view only the first few rows using `df.head`,
- view only part of the rows using `df.iloc`,
- use the `df.value_counts` method to count occurrences 
    of values in a certain column,
- use the `df.rename` method to rename a column,
- use the `pd.merge` function to join tables,
- use the `df.drop` method to remove certain columns,
- use the `df.set_index`, `df.sort_index` methods and `df.index.names` field
    to manipulate the index of the table,
- use the `df.pivot_table` method to create a summary from certain values using
    the `"mean"` and `"std"` aggregation functions,
- use `MultiIndex` tables (generated by the `pivot_table` method),
- export a `DataFrame` using the `to_csv` method,
- use the `pd.melt` function to create tall-skinny tables from short-fat tables,
- use the `df.describe` method to generate basic statistics for the table,
- used `df.plot.hist` to create histograms of `Series` objects
